🔍 Catalog Management: Easily organize and categorize books, periodicals, and other materials. Add, edit, or delete items, and keep track of their availability.

📋 Borrower Management: Maintain a database of library members, track their borrowing history, and manage fines and due dates seamlessly.

📅 Check-in and Check-out: Simplify the borrowing process with automated check-in and check-out. Send notifications for overdue items and manage renewals effortlessly.

📊 Reports and Analytics: Gain insights into library usage patterns, popular items, and borrower behavior. Make data-driven decisions to enhance your collection and services.

🔒 Admin and Staff Accounts: Set up different access levels for administrators and staff. Ensure data security and assign responsibilities accordingly.

💻 Technologies Used: This project is built using Java8, Spring Boot, Spring Security, Spring Data JPA, MySQL, Thymeleaf, Maven, JavaScript/CSS/HTML.

🤝 Contributions: We welcome contributions from the open-source community. Feel free to submit bug reports, feature requests, or pull requests to help us improve and grow the system together.


To run project:
1.Clone the repository to your local machine.
2.Run ./mvnw spring-boot:run.
